Drop these files into your Next.js repo (App Router). If you don't have App Router enabled, upgrade to Next.js 13.4+.

Structure:
- app/layout.tsx
- app/page.tsx (Home)
- app/about/page.tsx
- app/services/page.tsx
- app/portfolio/page.tsx
- app/testimonials/page.tsx
- app/contact/page.tsx
- app/sitemap.ts, app/robots.ts
- components/Navbar.tsx, components/Footer.tsx
- lib/site.ts
- styles/globals.css
- public/ (place images/logo here)

Run:
npm install
npm run dev
